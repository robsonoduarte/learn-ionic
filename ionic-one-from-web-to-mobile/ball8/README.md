Section 5 - The 8 Ball App
--------------------------------------

### To Run App

***Clone Repoexit***
```
git clone https://github.com/robsonoduarte/learn-ionic
```

***Run app in Browser***
```
cd learn-ionic/ionic-one-from-web-to-mobile/ball8/
ionic server
```

***Run app in Android ***
```
cd learn-ionic/ionic-one-from-web-to-mobile/ball8/
ionic platform android
ionic build android
ionic run android
```
